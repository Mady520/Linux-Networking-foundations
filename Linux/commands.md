## Structure 
Command name + [option/s] + [Arguments]

Options are case sensitive but can be combined.

## Linux Basic Commands 

1.cat --> Displays the contents of a file.
2.ls --> Lists files and directories.
  ls -l --> Long format of a directory/file with every detail along with modification time.
ls -lu --> Shows last accessed time. 
3.ncal --> calender with present day coloured.
  e.g ncal 2026 --> Will show claneder of full year. 
4.man --> shows manual pages/documentation of a command. 
  e.g man cat --> will display usage + every option of this command that can be used.
5.pwd --> Print working directory.   
6.cd --> Change directory 
  cd .. --> one directory back 
  cd 'directory name or folder name' 
  cd / --> Root directory. 
  cd ~ --> Hoome directory.
  cd /home/user/Document --> Absolute path. 
    cd /Document --> Relative path.
 8. mkdir --> Creates directory.
   mkdir (directory/folder name ) --> Creates the directory with the name given.
 9. touch --> Change file timestamps (modification, access time).
 10.file --> Determine the file type (by doing some test not based on the extension).
