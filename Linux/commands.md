## Structure 
Command name + [option/s] + [Arguments]

Options are case sensitive but can be combined.

## Linux Basic Commands 

1. `cat` &rarr; Displays the contents of a file.  
2. `ls` &rarr; Lists files and directories.  
3. `ls -l` &rarr; Long format of a directory/file with every detail along with modification time.  
4. `ls -lu` &rarr; Shows last accessed time.   
5. `ncal` &rarr; calender with present day coloured.  
     e.g `ncal 2026` &rarr; Will show claneder of full year.   
6. `man` &rarr; shows manual pages/documentation of a command.  
     e.g `man cat` &rarr; will display usage + every option of this command that can be used.  
7. `pwd` &rarr; Print working directory.  
8. `cd` &rarr; Change directory.  
9.  `cd ..` &rarr; one directory back.  
10. cd 'directory name or folder name'.  
11. `cd /` &rarr; Root directory.  
12. `cd ~ ` &rarr; Hoome directory.  
13. cd /home/user/Document &rarr; Absolute path.  
14. cd /Document &rarr; Relative path.  
15. `mkdir` &rarr; Creates directory.  
16. `mkdir (directory/folder name )` &rarr; Creates the directory with the name given.  
17. `touch` &rarr; Change file timestamps (modification, access time).  
18. `file` &rarr; Determine the file type (by doing some test not based on the extension).
19. `rm` &rarr; Remove files or Directories.
20. `rm -d` &rarr; for removing directories.
21. `rm -r` &rarr; Removes directories with their contents.
22. `cp` &rarr; Copies files and Directories.
23. `mv` &rarr; Moves files and Directories.
24. `rmdir` &rarr; removes empty Directories.
25. `clear` &rarr; used to clear the terminal screen.
26. `echo` &rarr; used to display text in terminal.
